# MonteMetrics

MonteMetrics replaces the single-number LTV model that most finance teams rely on with a probabilistic alternative that shows the full range of possible futures. Built on Spotify's real FY2025 SEC filings, it runs 10,000 Monte Carlo simulations to produce P10, P50, and P90 outcomes — revealing a $136 gap per subscriber that traditional models hide entirely. The goal: show that financial modeling should quantify uncertainty honestly, not pretend it does not exist. Every input is sourced from an official filing. Every output is traceable to real data.

## Required Tools
* Python:pandas, numpy, matplotlib, scipy, OS
* yfinance

## Tools and Technologies learnt/used in making this project
* Python: Core Language 
* yfinance: Pulling live spotify market data
